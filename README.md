# 📌 Image Compression using SVD

## 🚀 Overview
This project demonstrates image compression using Singular Value Decomposition (SVD).  
It reduces image size while maintaining visual quality.

## 🧠 How it works
- Convert image into matrix form  
- Apply SVD decomposition  
- Keep top important values  
- Reconstruct compressed image  

## 🛠️ Tech Used
- Python  
- NumPy  
- Matplotlib  
- Jupyter Notebook  

## 📷 Results
- Original vs Compressed images  
- Reduced size with minimal quality loss  

## ▶️ How to Run
1. Open notebook in Jupyter  
2. Run all cells  

## 🎯 Purpose
Learn SVD and apply it to real-world image compression.
